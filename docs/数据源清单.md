# TNFD 开源数据源清单

> 最后更新：2026 年 4 月 18 日  
> 用途：TNFD LEAP 框架自动化披露平台的数据源参考

---

## 核心数据源（优先级最高）

### 1. ENCORE 数据库（TNFD 官方推荐）

| 项目 | 详情 |
|------|------|
| **官网** | https://encorenature.org |
| **数据下载** | https://encorenature.org/ENCORE_DataFiles_Oct-2025.zip（最新版） |
| **归档版本** | https://encorenature.org/ENCORE_DataFiles.zip |
| **许可协议** | CC BY-SA 4.0（免费商用，需署名） |
| **内容** | 25 种生态系统服务 × 各行业的依赖度/影响度矩阵 |
| **用途** | **Evaluate 阶段**的核心数据源——行业特征匹配 |

**数据文件说明**：
- `ENCORE_DataFiles.zip` 包含完整的知识库数据
- 含 CSV 格式的行业 - 生态系统服务依赖矩阵
- 含 CSV 格式的行业 - 生态系统影响矩阵
- 可直接转换为 JSON 供前端使用

---

### 2. WDPA 保护区数据（全球保护区）

| 项目 | 详情 |
|------|------|
| **官网** | https://www.protectedplanet.net |
| **数据获取** | 需注册后下载（免费） |
| **格式** | Shapefile / GeoJSON |
| **内容** | 全球自然保护区边界矢量数据 |
| **用途** | **Locate 阶段**的空间叠加分析——判断资产是否位于/邻近保护区 |

**注意事项**：
- 需注册 IUCN 账户后下载
- 数据每月更新
- 中国区域数据可能不完整，需配合本土数据源使用

## 中国本土数据源

### 3. 国家生态数据中心（NESDC）

| 项目 | 详情 |
|------|------|
| **官网** | http://nesdc.org.cn |
| **数据获取** | 开放下载（部分需申请） |
| **内容** | 中国植被、生态系统、碳通量等观测数据 |
| **用途** | 中国区域的生态系统数据补充 |

**核心数据集**：
- 中国植被数据产品集（CVP）
- 中国陆地生态系统干扰数据集
- 生态系统碳氮水通量数据论文专辑
- 中国通量观测研究网络（ChinaFLUX）20 周年专题数据集
- 中国及世界作物种植数据（2000-2021 年）
- 青藏高原高寒草地土壤碳库及其动态变化

---

### 4. 地理遥感生态网（GISRS）

| 项目 | 详情 |
|------|------|
| **官网** | http://www.gisrs.cn |
| **联系方式** | gisrscn@126.com / QQ: 416498894 |
| **数据格式** | 矢量 / 栅格 |
| **内容** | 土地利用、植被指数、自然保护区等空间数据 |
| **用途** | 中国区域的保护区、植被、水系数据 |

**核心数据集**：
- 自然保护区分布数据
- 土地利用遥感监测数据
- 全国 NDVI 植被指数分布数据
- 全国水系流域空间分布数据集
- 全国行政区划边界数据
- 地形地貌数字高程 DEM 数据
- 土壤理化性质分布数据
- NPP 净初级生产力数据

### 5. 生态保护红线数据

| 项目 | 详情 |
|------|------|
| **来源 1** | 生态环境部 https://www.mee.gov.cn/ywgz/zrstbhc/stghjh/ |
| **来源 2** | 国家生态数据中心（部分省份） |
| **来源 3** | 地理遥感生态网（近似数据） |
| **限制** | 完全准确的矢量数据不公开 |
| **替代方案** | 用国家级自然保护区矢量数据近似 |

**注意事项**：
- 生态保护红线矢量数据未完全公开
- 可用「国家级自然保护区 + 国家公园 + 森林公园」近似替代
- 建议在报告中标注数据来源及精度限制

## 其他推荐数据源

### 6. WRI Aqueduct（水风险）

| 项目 | 详情 |
|------|------|
| **官网** | https://www.wri.org/aqueduct |
| **数据获取** | 免费注册后下载 |
| **格式** | 栅格 / 矢量 |
| **内容** | 全球水资源压力评估数据 |
| **用途** | **Locate 阶段**的水风险图层 |

**核心指标**：
- 水基线压力（Baseline Water Stress）
- 年内水变异度（Interannual Variability）
- 季节水变异度（Seasonal Variability）
- 地下水压力表（Groundwater Table）
- 水污染风险（Return Flow Ratio）

---

### 7. IPE 蔚蓝地图（中国污染数据）

| 项目 | 详情 |
|------|------|
| **官网** | https://www.ipe.org.cn |
| **API 文档** | https://www.ipe.org.cn/api |
| **数据格式** | JSON / CSV |
| **内容** | 企业污染排放、环境监管记录 |
| **用途** | **供应链环境风险排查** |

**核心数据集**：
- 全国企业环境信用记录
- 全国污染源监管信息
- 全国地表水水质监测数据
- 全国空气质量监测数据
- 全国固体废物污染环境防治信息
- 全国环评审批信息

**API 示例**：
```bash
# 搜索企业环境违规记录
curl "https://api.ipe.org.cn/v1/companies?keyword=企业名称"

# 获取企业详情
curl "https://api.ipe.org.cn/v1/companies/{id}/violations"
```

## 开发建议

### Day 0 数据准备清单

| 数据 | 来源 | 格式 | 处理 |
|------|------|------|------|
| ENCORE 行业矩阵 | encorenature.org | ZIP → CSV/JSON | 解压后转换为前端可用格式 |
| 中国保护区 | WDPA / GISRS | Shapefile → GeoJSON | 裁剪到中国范围，抽稀简化 |
| 生态红线 | 近似替代 | - | 用国家级自然保护区矢量数据近似 |
| 水风险 | WRI Aqueduct | 栅格 → 矢量分级 | 按风险等级转矢量 |
| 基础地图 | 高德/百度 | 瓦片 API | 前端直接调用 |

### 数据处理流程

```
1. 下载原始数据 → 2. 格式转换 → 3. 空间裁剪 → 4. 抽稀简化 → 5. 放入 public/data/
```

### 注意事项

1. **许可合规**：所有数据源均需遵守相应许可协议，报告中需标注数据来源
2. **数据精度**：开源数据精度有限，生成的报告应标注「仅供参考，建议人工复核」
3. **更新机制**：定期（季度）检查数据源更新，同步最新版本
4. **缓存策略**：大数据集（如 WDPA）建议本地缓存，避免频繁下载
